致木绵

发表于 2019-03-28 | 更新于 2019-03-29 | 分类于 诗歌
本文字数： 350 | 阅读时长 ≈ 1 分钟

偶然在豆瓣读到了致橡树,很欣赏舒婷老师的爱情观，在下面评论发现了一位网友的致木棉，个人也挺喜欢的，虽然现在社会各种狗血爱情观念不断地对我们进行洗脑，但我还是向往着这种爱情观。

致木棉

我如果爱你
绝不像萦绕的藤萝
没完没了的缠绕者你
我如果爱你
绝不像山间的小溪
吝啬的给予和无情的远离
也不止林鹊
整日满口甜言蜜语
也不止像萤火
微弱的光芒照不亮你的美丽
甚至露珠
直至春泥
不，这些都不够
我必须是你身旁的一株橡树
用不可羁绊的力量守护者你
根，盘旋在地下
叶，相融于一体
每一场雨里
我们都弹奏着爱的歌曲
但没有人
听懂我们的言语
你有你红硕的花朵
像霞 像玉
也像火炬
我有我强壮的躯杆
如钢铁般坚韧无比
又如俊峰般威仪
我们面对病痛 烈阳 霹雳
我们共享朝霞 繁星 春雨
这才是真正的爱情
不渝就在这里
不仅爱你美好的外表
更爱你坚强的个性
脚下的土地
